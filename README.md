# M-Charts
基于Canvas的Charts实现的技术预研

## DEMO
[在线预览](https://bojue.github.io/M-Charts)

![Demo](src/assets/demo/demo.png)

## 下载

```
git clone https://github.com/bojue/M-Charts.git

cd M-Charts

npm i
```

## 运行

```
npm run dev

```
